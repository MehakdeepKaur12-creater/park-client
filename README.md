## Parks (Client)

## Description
This is a React (Next.js) application used to manage parks. Users can view, add, update, and delete parks, and also add reviews for each park.

## Features
- View all parks
- Add a park
- Edit a park
- Delete a park
- Add review

## Technologies used
- Next.js 
- TypeScript
- Tailwind CSS

## Dependencies
1. npm install 
2. npm run dev 
3. Open in browser: http://localhost:3000

## API- This client connects to:
http://localhost:3001/api/parks

## Enviornment Variable
NEXT_PUBLIC_API_URL= http://127.0.0.1:3001
